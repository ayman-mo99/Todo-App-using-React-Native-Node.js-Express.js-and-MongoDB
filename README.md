# TodoApp \
A simple Todo App to practice on Express framework and use CURD operation in MongoDB \
##Technologies\
*Node v12\
*Express 4.x\
*MongoDB (MongoDB Atlas)\
*React Native (Expo 36)\
##Launch\
To run the server side\
`npm run server`\
To run the client side\
`npm run client`\
